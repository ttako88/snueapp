# 지도안 근거 데이터

여기에 CSV 를 넣으면 지도안 생성 품질이 올라간다.
**비어 있어도 지도안 생성은 동작한다** — 있으면 좋아지는 것이지 전제조건이 아니다.

넣을 파일과 형식: `docs/LESSON_DATA_CONTRACT.md`

```
성취기준_국어.csv   성취기준_수학.csv   …
단원구성.csv        ← 핵심. "5학년 국어 4단원" 을 성취기준에 잇는다
평가기준.csv
모형전개.csv
```

넣고 나서:

```bash
npm run lesson:check
```

몇 행이 통과·탈락했는지, 탈락은 몇 번째 줄에서 왜인지 찍어 준다.

> 이 폴더의 CSV 는 교육과정 문서·교과서 지도서에서 옮긴 것이다.
> 원문을 바꾸지 않는다. 모르는 값은 비워 둔다 — 추측해서 채운 값이
> 빈 칸보다 나쁘다.
