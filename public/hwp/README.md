# HWP(.hwpx) 서식 템플릿 폴더

지도안을 **네이티브 한글(.hwpx)**로 내보내기 위한 "검증된 서식" 템플릿을 여기 둔다.
Claude가 이 템플릿을 fetch → 표/문단 markup을 복제해 내용만 주입 → .hwpx 생성.
(한컴 없이 순수 코드로 만들려면, 한글에서 **실제로 열리는** 서식이 기준으로 필요.)

## 소유자가 만들 파일 (한글에서)
`base.hwpx` — 아래 3가지를 담은 **간단한 문서** 하나면 충분(첫 검증용):
1. 제목 한 줄 (예: `지도안`) — 굵게
2. 일반 문단 한 줄 (예: `학습목표: ...`)
3. **표 1개** (예: 3~4칸 × 2줄. 머리행 + 데이터행 1줄) ← 이게 제일 중요.
   본시학습 표 markup을 이 표에서 그대로 복제하므로, 표가 꼭 있어야 함.

### 저장 방법 (한글)
- 파일 > 다른 이름으로 저장 > 파일 형식을 **`한글 문서 (*.hwpx)`** 로 선택
- 저장 위치: 이 폴더 `snue-app/public/hwp/base.hwpx`

### 만든 뒤
Claude에게 "base.hwpx 넣었어" 라고 알려주면:
- 압축 해제해 header/section markup 학습 → HWPX 필러 작성 + "한글(.hwpx)" 버튼 배선
- 소유자는 다운로드한 .hwpx를 **한글에서 열어 정상 여부만 확인** → 되면 확정, 안 되면 오류 알려주면 교정

## (선택) 다음 단계 — 진짜 지도안 서식
첫 검증(base.hwpx)이 되면, 실제 **약안/세안 서식**(제목·학년·교과·단원·차시·학습목표·
본시학습표·평가)을 `약안.hwpx` / `세안.hwpx` 로 만들어 주면 서식 그대로 채워 뽑을 수 있음.

## 주의
- 특정 출판사 지도서 서식을 복제하지 말 것(저작권) — 표준/자작 서식 사용.
- .hwpx 는 공개 정적 파일로 서빙됨(빈 서식이라 무방).
