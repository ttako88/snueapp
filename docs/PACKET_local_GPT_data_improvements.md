## F. 수집 균형 + 범위·방법 (★ 미래엔 쏠림 교정 — 소유자 지시)
1. **출판사 균형**: 지금 미래엔에 심하게 쏠려 기형적(미래엔 239단원 vs 천재 4·아이스크림 13·지학사 24·비상 25·YBM 28). 비상·아이스크림·지학사·천재·YBM을 **골고루** 끌어올린다. 목표는 출판사별 균형.
2. **확보 대상 = 목차뿐**: 각 교과서의 **목차(단원명·차시명·총차시)** + 성취기준 코드 매핑. **교과서 '본문 페이지'가 아니다.** `단원구성.csv` 컬럼만 채우면 된다 → 몇백 페이지 캡처·OCR은 **불필요**.
3. **방법 = 목차 전사(다운로드·OCR 아님)**: 각 출판사 공개 e-교과서/미리보기 뷰어의 **'차례(목차)' 화면에 보이는 차시 제목 텍스트를 전사**한다. 뷰어가 목차를 색인/차례 데이터로 갖고 있으면 그 '제목 목록 텍스트'를 가져온다(메타데이터 전사). **다운로드 버튼이 없다고 그 교과서를 건너뛰지 말 것** — 필요한 건 목차 텍스트뿐이고 화면에 보인다.
4. **하지 말 것**: 본문 페이지 이미지·PDF를 접근제한 우회로 추출·복제하는 것. 필요도 없고(목차만 쓰므로) 교과서 본문은 출판사 저작물이라 유료 서비스에 리스크. 성취기준·평가기준은 교육부 공개 자료라 무방하지만 본문은 아니다.
5. 우선순위(단원 0/희박부터): 체육·통합 → 실과·미술·도덕·음악 → 저커버 출판사 교과.
6. 규칙 유지(§정합성): 성취기준 코드는 성취기준_*.csv에 실재하는 것만, 출판사·교과서ID 필수, 지어내기 금지(보이는 목차 원문만 전사).

## G. 지도서 추가 수집 (적당히·구조 중심 — 목차 위에 얹기)
목차는 '뼈대'만 준다. 지도서에서 **사실·구조 층**을 얹으면 생성 품질이 오른다. 단 **지도서는 출판사 저작물**이라 통째 복제는 금지 — 사실/구조는 추출 OK, 발문·활동 문장의 대량 verbatim 복사는 금지(요약·구조화로 옮긴다).
- **왜 필요**: 성취기준↔차시 매핑은 학생 교과서 목차엔 없고 **지도서에 있다** → `단원구성.csv`의 성취기준코드 칸을 채우는 핵심(사실).
- 차시(또는 단원)당 추출: ①관련 성취기준 코드(실재 코드만) ②적용 수업모형(10모형 키 매핑) ③시간배분(도입·전개·정리, 있으면) ④학습목표 1줄 ⑤주요 활동 1~2줄 **요약**(원문 통째 복사 금지) ⑥평가 방법/유의점 1줄.
- 발문 예시: 단원마다가 아니라 **수업모형당 대표 1~2개만**(참고용). **공개 지도안(에듀넷 등) 우선**, 지도서에서는 소수 예시만.
- **금지**: 지도서 페이지 전체·활동 전문·학습지/평가지 원본을 그대로 복사·저장. 필요한 건 '구조 + 소수 예시'이지 본문 재현이 아니다.
- 산출: 성취기준코드→`단원구성.csv`, 발화→`모형전개.csv`. 스키마에 없는 필드(학습목표·시간배분 등)의 저장처는 **소유자와 상의 후** 신설.

## H. 수집 실무 판단 규칙 (OCR·총차시·합본 — 진행 막힘 해소)
GPT가 지나치게 보수적으로 멈추는 지점을 명확히 한다.
- **차례(목차) 이미지는 OCR해도 된다.** 목차/차례가 이미지형이어도 **제목 리스트(단원명·차시명)를 OCR로 전사**하는 건 허용 — 이건 사실(메타데이터) 전사이지 본문 재현이 아니다. **이미지형이라고 건너뛰지 마라.** (§F의 "OCR 불필요"는 텍스트형 목차 전제였다. 이미지형 목차는 OCR로 전사.)
  - 단 **본문 페이지·활동 전문·학습지·평가지**를 OCR/추출하는 건 여전히 금지(저작권).
- **총차시 확정 규칙**:
  - **완전한 목차**(그 학년·교과서 차시가 전부 보임) → 총차시 = **나열된 차시 수(사실, 추정 아님)**, 차시번호 = 나열 순서.
  - **부분/샘플**("1단원 샘플만 제공" 등) → 확정 불가. **보류표에 원문 기록 + 총차시 미확인 표시**. 항목 수를 총차시로 둔갑 금지(← GPT가 맞게 함, 유지).
  - **합본 목차(3~4학년 등)** → 목차가 학년 구분을 표기하면 학년별로 쪼갠다. 구분이 없어 애매하면 학년별로 분리된 화면을 찾거나 보류.
  - 총차시가 목차에 없으면 **공개 연간지도계획·진도표**(학교·교육청·출판사 공개)에서 확인 시 그 값으로 채움(근거 남김).
- **보류표(pending)**: 식별자·목차 원문(이미지면 OCR로 전사한 텍스트)을 남겨, 나중에 총차시 보이는 공식 화면과 결합. GPT의 현행 보류 방식 OK — 단 이미지 목차는 **건너뛰지 말고 OCR 전사 후 보류**.
