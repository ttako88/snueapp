## A. 반드시 답이 필요한 것 (8개)

### 1. 지망·교환 어디까지?
`design.md` 21.1 — 실습학교 배정 **금전거래가 이미 공공연**하다(나무위키).
수요는 확실하나 실습학교는 학점·임용점수와 직결돼 기숙사 거래보다 리스크가 크다.

- **A** 지망 전 경쟁률 여론조사만
- **B** A + 학교 정보·교통·후기
- **C** B + 배정 후 "교환 원함" 매칭 (돈 없이 의사만) ✅
- **D** C + 금전 거래 중개

> C 추천 이유: 돈이 오가는 순간 학교 징계·법적 책임이 **상호님께** 갑니다.

### 2. 실습학교 평가에 지도교사 개인을 넣나?
한 학교 실습생이 20~30명이라 "3반 담임"이면 **특정됩니다.**
강의평가는 k=10 익명성이 되지만 여기선 안 됩니다.

- **학교 단위만** ✅
- 교사 개인도 포함

### 3. AI 지도안 — 일일 비용 상한
버그·어뷰징 한 번이면 상호님 카드가 긁힙니다. 원가 ≈ 200원/회.

- **하루 5,000원(≈25회) 넘으면 자동 차단 + 알림** ✅
- 다른 금액: ______

### 4. 결제 수단 — 지금 되나?
현금 500원을 받으려면 PG사 + **사업자등록**이 필요합니다.

- **광고 리워드 포인트로만 시작.** 현금 결제는 코드까지 만들어두고 스위치만 OFF ✅
- 사업자·PG 이미 있음 → 바로 붙임

### 5. 포인트 화폐 이름
- **분필** ✅ (교대답고 "분필 100개"가 자연스러움)
- 새록 / 씨앗 / P / 기타: ______

### 6. 출발지를 어떻게 받나? (최소환승 계산용)
- **지하철역·정류장 단위만** 저장 ("숭실대입구역") ✅
- 상세 주소까지

> 역 단위면 환승 계산에 충분하고 개인정보 부담이 확 줄어듭니다.

### 7. 실습모드 진입
- **별도 탭 + 실습 기간엔 자동으로 앞에 노출. 1학년은 숨김** ✅
- 상시 메뉴로 항상 노출

### 8. 자료(세안·약안) 거래
- **포인트 공유만.** 본인 작성분만 업로드 + 신고 기능 ✅
- 현금 거래 포함

> 남이 만든 지도안 재판매는 저작권 책임이 상호님께 갑니다.
