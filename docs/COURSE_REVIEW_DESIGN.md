## 한 줄 결론

**1단계 = "익명·저위험 강의 후기"로 좁혀 출시.** 작성자 공개속성 0개, 통계는 10명부터,
시험문제 원문 금지, 게시 후 덮어쓰기 대신 정정본·철회.

에타를 그대로 베끼지 않는 이유: 서울교대는 **2,700명·13심화과정**이라 수강생 10~20명 과목이
흔하다. 에타 방식(학과·학년 표시)을 그대로 쓰면 **누가 썼는지 사실상 특정**된다.

## 1. 작성자 익명성 (가장 중요)

**공개 속성 = 0개.** 아래를 화면·API 응답 어디에도 넣지 않는다.

| 공개 가능 | 공개 금지 |
|---|---|
| "수강 인증 완료" 표시 | 학과·심화과정, 학년, 분반 |
| 강의명·담당 교수 | **정확한 수강학기**, 정확한 작성일시 |
| 넓은 최신성("최근 1년 이내") | 시간표·성적 |
| 익명 후기 본문 | 익명 번호·작성자별 고정 별칭 |

- 내부에는 중복작성 방지·수강검증용으로 `member_id`·`offering_id`·`semester`를 보관하되
  **일반 조회 응답에 절대 미포함** (헌장 §3 식별자 분리).
- **성적 확정 이후에만 공개**하고, 즉시 공개가 아니라 **일정 시간 후 묶어서** 공개한다
  ("수업 직후 쓴 사람"이 추론되지 않게).
- 자유서술은 k값만으로 재식별이 사라지지 않는다 → 작성 전 경고 필수:
  *"본인을 특정할 수 있는 과제·발표·사건·날짜를 적지 마세요."*

## 2. 통계 최소 표본 (k=10, 헌장과 동일)

| 유효 평가 수 | 공개 방식 |
|---|---|
| 1~4 | 개별 후기만. 평균·최빈값·순위 **없음** |
| 5~9 | "표본이 적습니다". 최빈 응답이 **60% 이상 & 3명 이상**일 때만 '초기 경향' 한 문장. 백분율·평균·타 교수 비교 금지 |
| 10+ | 항목별 분포 그래프·평균 공개 |
| 20+ | 강의 간 비교·정렬 사용 가능 |
| 30+ | 외부 공개 통계 사용 가능 |

- **교수 평점 랭킹은 1단계에서 제외.** 도입하더라도 단순 평균 금지, 표본 보정
  베이지안 평균 사용.

## 3. 수정·삭제 정책 — "절대 불가"는 채택하지 않음

에타의 "수정·삭제 불가"는 어뷰징 방지엔 좋지만 **개인정보보호법상 정정·삭제 요구권**과
충돌한다. → **불변 이력 + 공개 철회**를 결합한다.

상태기계:
```
draft → submitted → published
                      ├→ corrected              (정정본이 새 버전, 구버전 비공개)
                      ├→ withdrawn_by_author    (작성자 철회)
                      ├→ hidden_by_moderation   (신고·정책위반)
                      └→ preserved_for_case     (사건 처리 목적 제한 보존)
                                  → purged      (보존기한 종료 파기)
```
- 한 **수강 건당 활성 평가 1개**
- 정정·재작성해도 **포인트 추가 지급 없음**
- 철회 시 지급 포인트를 **원장의 반대 거래로 회수**
- 무사건 철회 자료는 짧은 유예 후 작성자 연결·본문 파기

## 4. 명예훼손 방어선 (1단계부터 필수)

⚠️ 한국은 **진실한 사실 적시도 명예훼손 쟁점**이 될 수 있다. "사실이면 허용" 규칙 금지.
근거: 대법원 2025. 5. 29. 선고 2024도14555 · 형법 제307조.

1. 구조화 평가는 자동 접수, **k=10 전 통계 비공개**
2. **자유서술은 pending → 운영자 검토 후 공개** (금지어 필터만으로 부족)
3. 금지: 욕설·비하·외모·사생활·건강·가족·정치성향·범죄 단정·출처 없는 소문
4. "차별했다" 류 사실 주장은 공개 대신 **신고센터로 유도**
5. 개인정보·협박·불법자료 신고는 **즉시 임시 숨김**
6. 일반 불만 신고 1건만으로 자동 숨김하지 **않음** (보복신고 방지)
7. 서로 다른 인증회원의 중복 신고 또는 운영자 판단으로 숨김
8. **교수·학교의 삭제 요청 접수창구** 마련 — 단 요청만으로 무조건 삭제하지 않고 동일 기준 검토
9. 숨김·복구·기각·신원조회 **전부 audit log**
10. 작성자 통지 + 이의제기 경로
11. 처리 목표: 긴급 24시간 / 일반 72시간

## 5. 시험정보 → "시험 경향·준비 팁" (족보 아님)

저작권법 제32조의 시험 목적 예외는 **시험 시행을 위한 복제**에 관한 것이지, 족보를
공중에 공개하는 면허가 아니다. 시험문제도 창작성이 있으면 어문저작물이 된다.

| 허용 | 금지 |
|---|---|
| 객관식/서술형 등 **형식** | 문제·보기·답안의 원문 또는 사실상 복원 |
| 대략적 문항 수, 시험 범위 | 사진·스크린샷·PDF·**첨부파일 일체** |
| 오픈북 여부, 시간 압박 정도 | LMS·교수 배포자료 복제, 정답지 |
| 교수자가 공개한 평가방식 | 타인 족보 재업로드, 진행 중인 시험 정보 |
| 본인의 공부 방법·조언 | "단어 몇 개만 바꾼 문제" |

## 6. 티켓(포인트) 이코노미

| 행동 | 포인트 |
|---|---:|
| 학생 인증 완료 | +15 |
| 강의평가 **승인·공개** | +20 |
| 시험 경향 **승인·공개** | +10 |
| 강의평 상세 잠금해제 | −5 |
| 동일 강의 재열람 | 0 |
| 도움됨 3개 달성 | +2 (후기당 상한 +6) |
| 하루 신규 획득 상한 | +40 |

- 잠금은 후기 1건이 아니라 **"과목+교수 페이지 전체"**에 적용, 최소 한 학기 재과금 없음
- **에타의 "최초 등록 2배 보너스" 폐기** — 허위 선점 유인이 큼
- 강의평가 보상 > 시험경향 보상 (확보해야 할 핵심 데이터가 안전한 후기이지 족보가 아니므로)
- **불변 원장**: 잔액 직접 UPDATE 금지, ledger 합산으로만 계산. `idempotency_key` 필수.
  음수 잔액 금지. 철회·중복·조작 시 반대 거래로 회수.
- 현금 구매·양도·환급·앱 밖 사용 금지. **포인트로 제재해제·인증·운영권한 구매 금지**(헌장).
