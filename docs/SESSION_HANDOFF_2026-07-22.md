## 4. 절대 규칙 — 하지 말 것 (N3 / L3)

- **운영 DB 초기화·마이그레이션 적용·flag 활성화·Cron 활성·시크릿 입력·고아 삭제 금지** (사전 승인)
- **동결된 001~009 마이그레이션 수정 금지** — 추가는 새 번호로만
- **채팅/로그/git에 절대 넣지 않는다:** 커넥션 문자열, 비밀번호, service_role 키, JWT,
  anon/publishable 키 **값**, 실명, 학번, 학번 HMAC, 개인 이메일, 게시글 본문, auth user ID
- **Claude는 API 키/토큰을 필드에 입력하거나 소유자 계정에 로그인하지 않는다** — 소유자
  authorization으로도 안 바뀜. Kakao REST 키+Client Secret은 Supabase 대시보드에만.
- `setup-prod-secret.mjs` / `setup-verify-secret.mjs` = TTY 전용, deny-list
- **샌드박스 install-script 승인 게이트 우회 금지**
- **유료 스크립트 조심** — `gen-lesson-samples.mjs`는 실비(유료 Gemini) 사용.
  이번 밤 `--help` 실수로 유료 배치가 돌 뻔했음(즉시 죽여 실비 0원). 지금은
  `--help`/모르는 플래그를 실행 전 거부하도록 고침. 그래도 유료 도구는 소스부터 읽고 돌린다.
