# 임용고시 기능 아이디어 — 임고생 자문용 (배포 OK)

> 교대생 앱(새록)에 붙일 **임용 관련 기능** 후보입니다. 임용 준비 중인 분들께
> "이거 있으면 쓸 것 같다 / 별로다 / 이런 게 더 필요하다"를 여쭙고 싶어요.
> 번호로 편하게 코멘트 주세요.

## 정보·데이터
1. **지역별 TO·경쟁률·합격선** — 교육청별 선발인원·경쟁률·합격 컷을 한눈에. 지역 고를 때 판단.
2. **임용 공고·뉴스 알림** — 교육청별 일정·공고 뜨면 푸시. 놓치는 것 방지.
3. **전형 점수 계산기** — 1차 컷·가산점·2차 환산점수 넣으면 예상 위치. "이 점수면 어디 지원?"
4. **지역 이동 정보** — 합격 지역 선택 시 집값·생활비·교통 참고(연고 없는 지역 지원 결정).

## 학습·관리
5. **임용 학습 플래너** — 시험일 역산해 과목별 커리큘럼·진도 관리. D-day 기반.
6. **오답노트 + 약점 분석** — 틀린 문제 누적 → 약한 영역 자동 리포트.
7. **교육학·전공 암기 카드** — 스페이스드 리피티션(망각곡선) 반복 암기.
8. **기출 아카이브 + 풀이 공유** — 연도·지역·영역별 기출 + 해설(SR 경제로 등록/열람).
9. **요약노트 공유 장터** — 교육학·전공 정리본 공유(본인 작성분, SR).

## 스터디·사람
10. **스터디 매칭** — 지역·전공·2차(수업실연/면접) 조건으로 스터디원 매칭.
11. **스터디 관리** — 진도·출석·자료·일정 한 곳. (오프라인 스터디 운영 보조)
12. **2차 영상 피드백** — 수업실연·면접 영상 올려 서로 피드백(비공개 그룹).
13. **합격자 멘토 Q&A** — 합격 선배에게 질문(무료 게시판 or 유료 컨설팅 매칭).
14. **합격 수기·인터뷰 아카이브** — 합격자 공부법·타임라인 모음.

## 실습 연계 (새록만의 강점)
15. **실습 이력 → 임용 포트폴리오** — 앱에 쌓인 실습 데이터(수업·성찰·지도안)를 자소서·면접 소재로 자동 정리. ★ 새록의 실습 아카이브(킬러기능)와 직결.
16. **수업실연 = 지도안 생성기 연계** — 임용 2차 수업실연 연습에 지도안 툴 재활용.

## AI 활용
17. **면접·자소서 답변 코치** — 예상 질문에 답변 첨삭·모범답안 참고.
18. **교육학 개념 챗봇** — 헷갈리는 개념 물어보면 근거와 함께 설명.
