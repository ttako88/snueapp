## 되돌린 것 / 실수한 것

정직하게 적습니다.

1. **경로 검증 결함** — 지도안이 아니라 인증 서류 쪽인데, 제가 만든 함수가
   "신청 123번인데 999번 폴더에 저장"하는 걸 막지 못했습니다. 다른 사람 서류와
   엮일 수 있는 구멍이었고, GPT가 잡아줘서 고쳤습니다. 지금은 시스템이 경로를
   직접 만들어서 애초에 틀린 값이 들어갈 입구가 없습니다.

2. **테스트가 거짓 실패를 냈습니다** — 처음에 19개 중 1개가 실패로 떴는데,
   확인해보니 제품은 멀쩡하고 **제 테스트가 틀렸습니다.** 파일이 없을 때
   빈 값이 올 거라 가정했는데 실제로는 오류 객체가 왔습니다. 고쳤습니다.

3. **잘못된 파일이 커밋됐습니다** — `--dump` 라는 이상한 이름의 파일이
   실수로 저장소에 들어갔습니다. 지웠습니다.

4. **학교 이름 오인식** — 협력학교 목록을 자동으로 뽑았는데 "강남서초"를
   학교로 잡았습니다. 실제로는 **강남·서초구라는 지역 이름**입니다.
   "초"로 끝난다고 학교로 본 제 실수고, 기록에 남겨뒀습니다.

5. **AI 비용 상한을 처음엔 잘못 만들었습니다** — 이게 제일 중요합니다.
   상호님 돈이 걸린 부분인데, 제가 만든 첫 버전은 **상한을 실제로 못 지켰습니다.**
   GPT가 구멍 셋을 찾았고 전부 진짜였습니다:

   - 두 사람이 동시에 요청하면 둘 다 "아직 여유 있네" 하고 통과합니다.
     ₩4,800 쓴 상태에서 ₩200짜리 둘이 동시에 오면 ₩5,200이 됩니다
   - 실제 비용이 예상보다 크면 그만큼 그냥 넘어갑니다
   - AI 호출은 성공했는데 기록만 실패하면, **이미 나간 돈을 시스템이 잊어버립니다**

   전부 고쳤습니다. 이제는 날짜별로 자물쇠를 걸어 요청을 한 줄로 세우고,
   예상액이 아니라 **그 요청이 낼 수 있는 최대 금액**을 미리 잡아둡니다.
   기록이 실패해도 그 금액은 계속 쓴 것으로 칩니다.
   불변식 검사 12개 전부 통과했습니다.

   **다만 하나는 아직 증명 못 했습니다** — "진짜로 동시에 여러 요청이 와도
   합계가 한도를 안 넘는가"입니다. 코드는 넣었지만 실제로 확인하려면
   DB에 적용한 뒤에 해야 합니다. 통과한 척하지 않고 미증명으로 남겼습니다.

## 상호님 판단이 필요한 것 — 3개

### 1. AI 지도안 생성 API 키

화면과 안전장치는 다 만들었는데 **AI 키가 없어서 아직 안 돕니다.**
부트스트랩 마법사를 만들 때는 어느 회사를 쓸지 안 정했던 터라 안 받았습니다.

제 추천은 **Gemini 3 Flash** (세안 1건 원가 ₩20, 한국어·교육 도메인 강함).
→ https://aistudio.google.com/apikey 에서 키 만드시고 `npm run setup` 다시 돌리면
그 항목만 물어보게 해두겠습니다. 5분이면 됩니다.

**비용 안전장치는 이미 다 넣었습니다**: 하루 총 5,000원 넘으면 자동 차단,
1인당 하루 1,000원, 한 번에 200원 넘는 요청은 아예 안 보냄. 예산 확인이
안 되면 호출 자체를 안 합니다.

### 2. 로그인 개방 (도메인 제한 폐기 + 일반 회원가입)

**코드는 다 준비됐습니다.** 스위치 한 줄과 DB 변경 하나만 하면 됩니다.

다만 **밤에 하지 않았습니다.** GPT 판정이 맞았는데, 이건 "누가 가입할 수
있는가"를 바꾸는 보안 경계 변경이고 가입 확인 메일이라는 외부 발송도
생깁니다. 제 야간 금지선에 걸립니다.

**낮에 상호님 깨어 계실 때 적용하겠습니다.** 그때 실제로 새 계정 만들어서
가입 → 로그인 → 인증 제출까지 확인하겠습니다.

### 3. 강의평가 (과목 1,267건)

어제 제가 승인 없이 운영에 넣은 그 데이터입니다. 롤백 안 하기로 하셨으니
그대로 두되, **GPT가 아직 "정식 데이터로 인정" 판정을 안 했습니다.**
강의평가 기능을 켜려면 그 판정과 016 마이그레이션 검수가 먼저입니다.
